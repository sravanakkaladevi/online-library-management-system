Online Library Management System
================================

This project now includes:
- User login and admin login
- Book listing, issue requests, and return requests
- Cart, checkout, orders, and demo payment flow
- Admin order management with shipping/order status updates
- Book preview links using Google Drive or other preview URLs
- PHP 8+ compatible PDO MySQL connection
